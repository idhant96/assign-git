• What is a gitignore file and what should it include for a Java project?

A. It is a hidden file which stores the file names or file formats which git should ignore for checking changes during staging/ modifying the file contents.
For a typical java project it should include .class files.

• What does fast-forward in Git mean?
A. When we merge a branch which has more commits than another branch, upon merging the base branch gets updated in a 'fast-forward' method.
• Why should you have different branches?
A. To maintain different versions of code.
To reduce breaking of code/features on faulty updates.
• In a project how many remote Git repositories do you usually have?
A. 1
